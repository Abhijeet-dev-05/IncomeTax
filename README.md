# 💰 Income Tax Calculator

A simple and user-friendly web application to calculate income tax based on user input.

## 🚀 Live Demo

👉 https://income-tax-puce.vercel.app/

## 📌 About the Project

This project is an Income Tax Calculator built using HTML, CSS, and JavaScript. It helps users quickly estimate their tax based on income values by applying predefined tax rules. The goal is to simplify tax calculation with a clean and interactive interface.

## 💡 Features

* 📊 Calculate tax based on income
* ⚡ Instant results
* 🎯 Easy-to-use interface
* 📱 Responsive design

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
